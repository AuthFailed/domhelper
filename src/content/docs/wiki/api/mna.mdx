## Содержание

Метод содержит в себе список партнерских провайдеров с типами подключений и авторизаций.
| Данные | Авторизация | Запрос |
|--------|-------------|--------|
| JSON | Нет | GET |

## Ссылка

`https://authfailed.github.io/domru-helper/api/mna.json`

## Пример ответа

```json
{
  "mna": [
    {
      "name": "Interzet",
      "authorization": "IPoE Static",
      "connection": "Ethernet",
      "link": "Ссылка на БЗ"
    }
  ]
}
```
