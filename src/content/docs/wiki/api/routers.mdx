## Содержание

Метод содержит в себе список роутеров со ссылками на инструкции по их настройке, базу знаний и эмуляторы.
| Данные | Авторизация | Запрос |
|--------|-------------|--------|
| JSON | Нет | GET |

## Ссылка

`https://authfailed.github.io/domru-helper/api/routers.json`

## Пример ответа

```json
{
  "routers": [
    {
      "Name": "TP-Link EC220",
      "PPPoE": "Ссылка на настройку PPPoE",
      "DHCP": "Ссылка на настройку DHCP",
      "IPoE": "Ссылка на настройку IPoE Static IP",
      "Channels": "Ссылка на настройку каналов Wi-Fi",
      "Settings": "Ссылка на интерфейс роутера",
      "BZ": "Ссылка на базу знаний",
      "Emulator": ["Ссылка на эмулятор", "Ссылка на эмулятор 2"]
    }
  ]
}
```
