## Содержание

В качестве сервиса для проверки IP используется https://ip-api.com <br/>
Принимаемые форматы:

- 1.2.3.4

| Данные | Авторизация | Запрос |
| ------ | ----------- | ------ |
| JSON   | Нет         | GET    |

## Ссылка

`http://ip-api.com/json/$[айпи_для_проверки]?fields=country,regionName,city,org&lang=ru`

## Пример ответа

```json
{
  "country": "Россия",
  "regionName": "Тюмень",
  "city": "Тюмень",
  "org": "JSC \"ER-Telecom Holding\" Tyumen' branch"
}
```

## Лимиты

45 HTTP-запросов с одного IP-адреса в минуту.<br/>
При превышении лимита сервер будет возвращать статус 429.
