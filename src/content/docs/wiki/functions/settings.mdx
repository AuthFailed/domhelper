import { Tabs, TabItem } from "@astrojs/starlight/components";

:::note
Перед изменением настроек убедись, что у тебя установлена последняя версия расширения
:::

<Tabs>

<TabItem label="Общие" icon="setting">
<details>
<summary>Проверять обновления</summary>
Функция проверяет наличие доступных обновлений расширения при запуске браузера со страницы [релизов](https://github.com/AuthFailed/domru-helper/releases)

**Состояние:**

- Включено - Расширение проверяет наличие обновлений (Стандарт)
- Выключено - Расширение не проверяет наличие обновлений

</details>

<details>
<summary>Темная тема</summary>
Функция включает темную тему для окна расширения и настроек.

**Состояние:**

- Включено - Темная тема включена
- Выключено - Темная тема выключена (Стандарт)

**Пример:**

- Темная тема включена
  ![Темная тема](../../../assets/settings-darktheme.png)
- Темная тема выключена
  ![Светлая тема](../../../assets/settings-lighttheme.png)

</details>

</TabItem>

<TabItem label="Генезис" icon="email">
<details>

<summary>Быстрые кнопки</summary>
Функция включает отображение быстрых кнопок в Генезисе.

**Состояние:**

- Включено - Кнопки отображаются (Стандарт)
- Выключено - Кнопки не отображаются

**Пример:
**![Быстрые кнопки в Генезисе](../../../assets/genesys_fastbuttons.png)

</details>

<details>
<summary>Скрывать бесполезные кнопки</summary>
Функция скрывает со страницы Генезиса следующие кнопки:
- Facebook in Progress
- Facebook Draft
- Twitter in Progress
- Twitter Draft
- Лого Генезиса
- Кнопка поддержки Генезиса

**Состояние:**

- Включено - Кнопки скрываются
- Выключено - Кнопки не скрываются (Стандарт)

</details>

<details>
<summary>Отображение аварийности НЦК2</summary>
Функция включает отображение статуса аварийности в Генезиса для НЦК2. Состояние аварийности берется из канала "Аварийка ОТРС".

**Состояние:**

- Включено - Аварийность отображается
- Выключено - Аварийность не отображается (Стандарт)

**Пример:
**![Отображение аварийности НЦК2](../../../assets/settings_otpc.png)

</details>
</TabItem>

<TabItem label="АРМ" icon="seti:odata">
<details>
<summary>Основные настройки</summary>
<details>
  <summary>Скрывать СПАС</summary>
  Функция автоматически сворачивает окно СПАСа на договоре. Окно **не удаляется**, есть возможность развернуть его обратно.

**Состояние:**

- Включено - СПАС сворачивается при открытии договора
- Выключено - СПАС не сворачивается (Стандарт)

</details>

<details>
  <summary>Отображать быстрые СМС</summary>
  Функция добавляет кнопки для быстрого выбора типа СМС:
  - Оплата
  - ЛК
  - PPPoE
  - StaticIP

**Состояние:**

- Включено - Кнопки добавляются при открытии договора (Стандарт)
- Выключено - Кнопки не добавляются

**Пример:**
![Быстрые смс](../../../assets/arm-fastsms.png)

</details>

<details>
  <summary>Помечать обращений как не ошибочные</summary>
  Функция автоматически помечает обращение как не ошибочное при его изменении.

**Состояние:**

- Включено - Обращения помечаются не ошибочными
- Выключено - Обращение никак не помечается (Стандарт)

**Пример:**
![Не ошибочное обращение](../../../assets/arm-wrongtransfer.png)

</details>

<details>
  <summary>Убирать галочку Назначить на меня</summary>
  Функция автоматически убирает галочку на поле **Назначить на меня** при изменении обращения.

**Состояние:**

- Включено - Галочка убирается
- Выключено - Галочка не убирается (Стандарт)

**Пример:**
![Назначить на меня](../../../assets/arm-setonme.png)

</details>

<details>
  <summary>Уведомлять об особом клиенте</summary>
  Функция проверяет и оповещает об особом клиенте при открытии договора.

**Состояние:**

- Включено - Расширение проверяет и оповещает об особом клиенте (Стандарт)
- Выключено - Проверка выключена

**Пример:**
![Особый клиент](../../../assets/arm-specialclient.png)

</details>

</details>

</TabItem>

<TabItem label="Линия" icon="rss">
<details>
<summary>Подсветка назначений</summary>
Функция подсвечивает специалистов в назначениях отдельными цветами.

**Состояние:**

- Включено - Назначения подсвечиваются
- Выключено - Назначения не подсвечиваются (Стандарт)

</details>

<details>
<summary>Кнопки дежурных</summary>
Функция добавляет полезные кнопки для дежурных:
- Очередь вопросов
- Заглушки
- Предложка
- Прогноз SL
- ТС 2.0
- Генератор ленты

**Состояние:**

- Включено - Кнопки добавляются на линию
- Выключено - Кнопки не добавляются (Стандарт)

</details>

<details>
<summary>Прогноз SL</summary>
Функция отображает прогноз SL на следующие 30 минут. Опция доступна только для специалистов с доступом к инструменту.

**Состояние:**

- Включено - Прогноз SL загружается и отображается
- Выключено - Прогноз SL не загружается (Стандарт)

</details>

<details>
<summary>Быстрые кнопки</summary>
Функция позволяет добавить быстрые кнопки в верхнюю строку линии.

**Состояние:**

- Включено - Выбранные в списке на странице кнопки добавляются (Стандарт: Почта, Обеды)
- Выключено - Кнопки не добавляются

</details>
</TabItem>

</Tabs>
