<TabItem label="Общие" icon="setting">
<details>
<summary>Проверять обновления</summary>
Функция проверяет наличие доступных обновлений для расширения.

**Особенности:**

- Расширение проверяет наличие новой версии в [релизах](https://github.com/AuthFailed/domru-helper/releases)
- Проверка происходит только при запуске браузера
- Перекачивать расширение не нужно, расширение предложит обновить само себя

**Состояние:**

- Включено - Расширение проверяет наличие обновлений (Стандарт)
- Выключено - Расширение не проверяет наличие обновлений

</details>

<details>
<summary>Темная тема</summary>
Функция включает темную тему для окна расширения и настроек.

**Состояние:**

- Включено - Темная тема включена
- Выключено - Темная тема выключена (Стандарт)

**Пример:**

- Темная тема включена
  ![Темная тема](../../../assets/settings-darktheme.png)
- Темная тема выключена
  ![Светлая тема](../../../assets/settings-lighttheme.png)

</details>

<details>
<summary>Скрывать бесполезные кнопки</summary>
Функция удаляет со страницы Генезиса следующие кнопки:
- Facebook in Progress
- Facebook Draft
- Twitter in Progress
- Twitter Draft
- Лого Генезиса
- Кнопка поддержки Генезиса

**Особенности:**

- Кнопки удаляются полностью. Для восстановления требуется отключить функцию и обновить страницу Генезиса

**Состояние:**

- Включено - Кнопки скрываются
- Выключено - Кнопки не скрываются (Стандарт)

</details>

<details>
<summary>Отображение аварийности НЦК2</summary>
Функция включает отображение статуса аварийности в Генезиса для НЦК2. Состояние аварийности берется из канала "Аварийка ОТРС".

**Особенности:**

- Статус аварийности обновляется каждые 5 секунд
- Статус берется из последнего сообщения в канале
- Статус отображает полный контент сообщения в хедере
- Если есть слово `вкл` или `он` - надпись будет красным. В ином случае зеленым

**Состояние:**

- Включено - Аварийность отображается
- Выключено - Аварийность не отображается (Стандарт)

**Пример:**
![Отображение аварийности НЦК2](../../../assets/settings_otpc.png)

</details>

<details>
  <summary>Отображать копирование адреса</summary>
  Добавляет кнопку на вкладку Инфо договора в АРМе для копирования адреса. Адрес копируется без индекса, индекс удаляется по Regex
  ```txt

  ```

**Состояние:**

- Включено - Кнопка копирования адреса добавляется на договор (Стандарт)
- Выключено - Кнопка не добавляется

</details>

<details>
  <summary>Копировать адрес без города</summary>
  Дополнение к предыдущей функции. Адрес копируется без индекса и улицы, улица удаляется по Regex
  ```txt
/.*?(ул\.)/i
  ```

**Состояние:**

- Включено - При копировании кнопкой удаляется город клиента
- Выключено - При копировании кнопкой город не удаляется (Стандарт)

</details>

<details>
<summary>Подсветка назначений</summary>
Функция подсвечивает специалистов в назначениях отдельными цветами.

**Особенности:**

- Подсвечиваются:
  - Проектная деятельность [Светлая тема - <span style="background-color:#F7DCB9">#F7DCB9</span>, Темная тема - <span style="background-color:#5D6D7E">#5D6D7E</span>]
  - РСГ [Светлая тема - <span style="background-color:#B3C8CF">#B3C8CF</span>, Темная тема - <span style="background-color:#4C688B">#4C688B</span>]
  - Обучение [Светлая тема - <span style="background-color:#DFCCFB">#DFCCFB</span>, Темная тема - <span style="background-color:#75608E">#75608E</span>]
  - Помощь смежному отделу [Светлая тема - <span style="background-color:#F3D0D7">#F3D0D7</span>, Темная тема - <span style="background-color:#82494A">#82494A</span>]
- Цвет автоматически меняется в зависимости от текущей темы Линии
- Подсветка автоматически обновляется при изменении DOM страницы

**Состояние:**

- Включено - Назначения подсвечиваются
- Выключено - Назначения не подсвечиваются (Стандарт)

</details>

<details>
<summary>Прогноз SL</summary>
Функция отображает прогноз SL на следующие 30 минут. Опция доступна только для специалистов с доступом к инструменту.

**Особенности:**

- Формат вывода:

<Card>
  Прогноз на *промежуток в 30 минут*

Прогнозный SL: _прогноз SL на 30 минут_\
Прогноз чатов: _прогноз чатов на 30 минут_\
Разница людей: _прогноз разницы людей на 30 минут_\
Нужно держать _требуемый сл на 30 минут_\
Прогноз SL: _SL через 30 минут_

</Card>
- Обновляется раз в 10 секунд

**Состояние:**

- Включено - Прогноз SL загружается и отображается
- Выключено - Прогноз SL не загружается (Стандарт)

</details>

<details>
<summary>Быстрые кнопки</summary>

**Состояние:**

- Выключено - Кнопки не добавляются
